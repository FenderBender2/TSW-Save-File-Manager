# TSW-Save-File-Manager 1.0.0

### 🚀 First Public Release

### New Features
- Auto‑detects installed TSW versions (3–6 so far)
- Scans all Steam library folders
- Extracts correct EXE icon for each version
- Watches the active save file and updates UI instantly
- Clean UI with version selector and icon preview

### Notes
TSW save files cannot be decoded (UE4 binary format).  
Enter meaningful file names to describe your save files.
